================================================================================
COLUMNS TO CREATE
================================================================================

1. TITLE (Already exists by default)
   - Type: Single line of text
   - Required: Yes
   - Description: Application name (e.g., "Outlook", "Teams")

2. APP ID
   - Display Name: App ID
   - Internal Name: AppId
   - Type: Single line of text
   - Required: Yes
   - Description: Unique identifier (lowercase, no spaces)
   - Example: "outlook", "teams", "power-bi"

3. DESCRIPTION
   - Display Name: Description
   - Internal Name: Description
   - Type: Multiple lines of text
   - Required: No
   - Number of lines: 3
   - Description: Brief description of the application

4. ICON
   - Display Name: Icon
   - Internal Name: Icon
   - Type: Hyperlink or Picture
   - Format: Hyperlink
   - Required: No
   - Description: Emoji (üìß) or URL to icon image

5. APP URL
   - Display Name: App URL
   - Internal Name: AppUrl
   - Type: Hyperlink or Picture
   - Format: Hyperlink
   - Required: Yes
   - Description: Link to the application

6. CATEGORY
   - Display Name: Category
   - Internal Name: Category
   - Type: Choice
   - Required: Yes
   - Default: Custom
   - Choices (one per line):
       Microsoft 365
       Productivity
       Communication
       Utilities
       Custom
       HR Systems
       Finance Systems
       External Services

7. IS EXTERNAL
   - Display Name: Is External
   - Internal Name: IsExternal
   - Type: Yes/No
   - Required: No
   - Default: Yes
   - Description: Whether app opens in new tab

8. DISPLAY ORDER
   - Display Name: Display Order
   - Internal Name: DisplayOrder
   - Type: Number
   - Required: No
   - Default: 100
   - Min: 0
   - Max: 1000
   - Decimals: 0
   - Description: Sort order (lower numbers first)

9. IS ACTIVE
   - Display Name: Is Active
   - Internal Name: IsActive
   - Type: Yes/No
   - Required: No
   - Default: Yes
   - Description: Show/hide this app

================================================================================
MAINTENANCE
================================================================================

To add new app:
   SharePoint list ‚Üí New ‚Üí Fill fields ‚Üí Save

To edit app:
   Find app ‚Üí Click to open ‚Üí Edit ‚Üí Save

To hide app:
   Edit app ‚Üí Set "Is Active" to No ‚Üí Save

To delete app:
   Select app ‚Üí Delete (or set Is Active to No)

To reorder apps:
   Edit Display Order numbers (lower = first)

To add category:
   List settings ‚Üí Category column ‚Üí Edit ‚Üí Add choice

================================================================================
TROUBLESHOOTING
================================================================================

Apps not showing?
   ‚Üí Check "Is Active" = Yes
   ‚Üí Verify list name is "Applications"
   ‚Üí Check user has Read permission

Icons not showing?
   ‚Üí Use emojis instead of URLs
   ‚Üí Verify image URL is accessible
   ‚Üí Use HTTPS URLs only

Permission errors?
   ‚Üí List settings ‚Üí Permissions
   ‚Üí Grant Read to "All authenticated users"
